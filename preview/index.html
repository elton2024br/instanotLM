Instagram Profile
Extractor → NotebookLM

Extraia dados completos de perfis publicos do Instagram e gere documentos otimizados para analise com IA no Google NotebookLM. Ferramenta para jornalismo investigativo.

Instalar Extensao Como Funciona

Pipeline de Extracao

4 etapas automatizadas — do perfil ao documento pronto para IA

👤

1. Perfil

Extrai nome, bio, seguidores, categoria e status via API v1 + GraphQL + DOM

📋

2. Posts

Coleta todos os posts com paginacao automatica: legendas, hashtags, metricas, imagens

🔍

3. OCR / Alt

Captura descricoes automaticas das imagens (accessibility captions) para contexto

📄

4. Documento

Gera Markdown otimizado para NotebookLM com estatisticas, indices e timeline

Funcionalidades

🔐

Sessao do Browser

Usa a sessao autenticada do seu Chrome — sem precisar fornecer credenciais

🛡️

Anti-Deteccao

Delays aleatorios, tratamento de rate limit (429), retry automatico com backoff

📊

Estatisticas Completas

Top posts, hashtags mais usadas, localizacoes, mencoes, timeline mensal

📑

Carrosseis

Suporte completo a carrosseis com todas as imagens extraidas e indexadas

🔄

3 Fallbacks de API

API v1 → GraphQL → DOM parsing. Se uma falhar, tenta a proxima automaticamente

💾

Dual Export

Gera Markdown otimizado para NotebookLM + JSON completo para uso programatico

Instalacao

1

Abra as Extensoes do Chrome

Digite chrome://extensions/ na barra de endereco

2

Ative o Modo Desenvolvedor

Toggle no canto superior direito da pagina

3

Carregue a Extensao

Clique em "Carregar sem compactacao" e selecione a pasta descompactada da extensao (onde esta o manifest.json)

4

Navegue ate um Perfil

Va ate instagram.com/{username} e clique no icone da extensao
